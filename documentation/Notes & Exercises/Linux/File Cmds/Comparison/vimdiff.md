## vimdiff

```bash
# Opens vim with multiple files in side-by-side view, one file in each window, w/ differences highlighted
vimdiff FILE1 FILE2
# Shortcuts:
# do		gets changes from the window into the current one
# dp		puts changes from the current window into the other
# ]c		jumps to the next change
# [c		jumps to the previous change
# :qa		Quit from all docs at once (otherwise you have to quit twice)
```
